# A股多智能体量化选股系统

## 系统概述

本系统采用 **"板块初筛 → 6专家并行选股 → 投票仲裁 → 风控审查"** 的四阶段流水线架构，通过多个风格互补的AI智能体协同工作，从全市场数千只A股中筛选出未来1~10个交易日具有爆发潜力的个股。

## 快速开始

### 方式一：纯AI对话模式
1. 将本目录下所有 `.md` 文件作为系统提示词加载
2. 对AI发送："请执行今日选股"
3. AI将按流程逐步执行并输出结果

### 方式二：Python + AI 混合模式（推荐）
```bash
# 1. 安装依赖
pip install akshare pandas

# 2. 基础数据采集
python data_fetcher.py

# 3. 将数据文件提供给AI进行板块初筛
# 4. AI确定核心板块后，深度采集
python data_fetcher.py --sectors "算力,机器人,创新药"

# 5. 将所有数据提供给AI，执行完整选股流程
```
